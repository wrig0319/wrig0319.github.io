  2. Qus:Define version control and give one example.

    Ans: A system that records changes made to a file or set of files so that users can view previous changes done.

  3. Qus:What tag is used to create the title of a web page? Give an example.

    Ans: <title>This is a title </title>

  4. Qus:4. What tag is used to create a paragraph in html? Give an example.

    Ans: <p>This is a sentence</p>

  5. Qus:5. Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans: <span> is different from <p>, <a>, <i> because it is used to group inline-elements in a document, and it does not provide any visual change by itself. It also provides a way to add a hook to a part of a text or another section in a document.
